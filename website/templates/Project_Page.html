About The Project

This project aims to solve the problem of abandoned student projects at Multimedia University. Every semester, brilliant systems are built and forgotten. The MMU OSSD acts as a directory to preserve these legacies and find the next generation of maintainers.

We are currently implementing the core owner management features and project administration modules.
